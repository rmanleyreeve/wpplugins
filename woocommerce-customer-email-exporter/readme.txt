A feature for WooCommerce allowing you to export a CSV list of customer emails from the admin panel.

== Description ==

WooCommerce Customer Email Exporter simply adds a extra link under the Reports > Customers tab to export all customer emails into a single CSV file.

= Features =

* Compatible with WooCommerce 3
* Downloadable CSV file.
* Filename with store name and download time.
* All customer emails (existing and new).

= Usage =

* The plugin requires [WooCommerce](http://wordpress.org/plugins/woocommerce/) 2.1 or greater to be installed and activated in order to work. To download the CSV, you must have installed and activated WooCommerce first.
* After successful installation, you will find the option `WooCommerce > Reports > Customers > Export Customer Emails` in the admin panel.
